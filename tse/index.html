TSE

TSE is a text editor included inside of TypeSet that was ported over to other window managers.



Installing

Please use the netbox-packages package manager to install TSE as it is the easiest way. We will need 2 packages to get this to work

root@netbox $ pmi tseport
====================================
Your package is finished installing!
====================================

root@netbox $ pmi tsethemes
====================================
Your package is finished installing!
====================================

Usage

Just use it as if you were using ef, but type tse instead!

root@netbox $ tse hello
TSE was opened in a new window.

The Advantages of TSE

Kedit and TSE are very different in how they render the text. The built in tools render HTML inside the document. Say you tried to edit nmConfig, well it would not work because it would render the elements. TSE dosent render them, it just writes out them as text, allowing you to edit it.
