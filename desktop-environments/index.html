Desktop Environments

A Desktop Environment is typically the GUI part of a operating system. Think of how linux has different DE's like Gnome, Xfce, and i3. Netbox has a few of its own. They are all free and open source for you to use. Here is a list of all of them (with pictures!)

#1 TypeSet (ts)

TypeSet was made for the customization freaks. It also has custom utilites built in!



#2 Nbwm

Nbwm is the default Window Manager that was created by the developers. All other DE's are based on it.



#3 Iwm

A modified version of Nbwm that has a clock and smaller window bar.



#4 Powerline

The origion of TypeSet with a powerline bar.



How to install a DE

Goto the downloads page and grab the specific DE you want. Unzip and drag the files into the folder that contains kernal.js. Then open the index.html file and you are ready to start learning how to use NetBox!
