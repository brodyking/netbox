Brew

Brew is a web browser that we talked about in Installing Apps. But this page will give a more in-depth tutorial on how to use it.



Installing

Please use the netbox-packages package manager to install brew as it is the easiest way.

root@netbox $ pmi brew
====================================
Your package is finished installing!
====================================

Usage

Using brew is very simple. Type in brew + any website url to view that website.



The config button brings you to a empty screen. This is because there are still no settings to modify in the latest version of brew. The goto button lets you change what site you are currently viewing.

Known Issues
