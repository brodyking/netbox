Installing Apps

Out of the box you are missing some components you might want. Mainly, a web browser. Thankfully there is one that I have created. Before we install the browser though, lets install a package manager.

Why do we need a package manager?

Having all of our packages in one place makes it really nice to install and update them. We will be using netbox-packages repository. It contains brew, our browser, and other apps you might like.

To install netbox-packages, type the following command into netbox. DO NOT COPY PASTE IT, IT WILL NOT WORK.

root@netbox $ install https://brodyking.github.io/netbox-packages/install.js
reboot your system and you should be good to go!

Using the package manager

The command to install packages is pmi [package name]. To install brew, type the following command. DONT COPY PASTE!

root@netbox $ pmi brew
====================================
Your package is finished installing!
====================================
After installing brew, type the following command to view my website!

root@netbox $ brew https://bkjs.us
Please remember some websites will not work because they are blocking iFrames.
